## Install

```sh
npm install
```

## Usage

Usage is defined by the [Gatsby CLI commands](https://www.gatsbyjs.org/docs/gatsby-cli/).

To build, use:

```sh
gatsby build
```

To run a live development server, use:

```sh
gatsby develop
```

## Run tests

```sh
npm run test
```
